## Explanation

Object pronouns replace noun objects to avoid repetition.

**Direct object pronouns** (replace the direct object):
| Person | Pronoun |
|--------|---------|
| me/m' | me |
| you | te/t' |
| him/it | le/l' |
| her/it | la/l' |
| us | nous |
| you (pl.) | vous |
| them | les |

**Indirect object pronouns** (replace à + person):
- *lui* (to him/her), *leur* (to them)

Object pronouns go **before the conjugated verb**:
- *Je le vois.* (I see him/it.)
- *Elle lui parle.* (She speaks to him/her.)

## Examples

- *Tu aimes ce film?* → *Oui, je l'aime.* (Yes, I like it.)
- *Il parle à Marie.* → *Il lui parle.* (He speaks to her.)

## Pitfall

In the passé composé with *avoir*, the past participle **agrees** with a preceding direct object: *Je l'ai vue.* (I saw her — *vue* agrees with *la*.)
